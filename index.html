Password Strength Checker

Teste a força da sua senha e aprenda como criar senhas mais seguras

Digite uma senha para começar

Critérios de Segurança:

❌ Pelo menos 8 caracteres
❌ Pelo menos uma letra minúscula
❌ Pelo menos uma letra maiúscula
❌ Pelo menos um número
❌ Pelo menos um caractere especial (!@#$%^&*)

Dicas para uma senha mais forte:

  • Use pelo menos 8 caracteres
  • Misture letras maiúsculas e minúsculas
  • Inclua números
  • Use caracteres especiais
  • Evite informações pessoais
  • Não use senhas comuns ou sequenciais

Exemplos:

12345678 🔓
Fraca
PxH1#n!8 🔒
Forte

⚠️ Nunca compartilhe suas senhas reais. Este é apenas um verificador educacional.
